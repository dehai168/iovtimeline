# IovTimeLine

基于canvas开发,支持多通道录像时间轴展示,当前时间点播放,拖动播放以及时间轴放大缩小,以满足行业应用。

## 使用

```js

   npm i iovtimeline --save

```

## 开发

### 技术栈

   es6

### 安装

   npm install

### 编译

   npm run build

### 运行

   npm run dev

### 发布

   npm publish
